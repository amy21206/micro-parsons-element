Tool 2 Tutorial:

The only difference between tool 2 and tool 1 is how you input the regex.
Input Blocks
Instead of writing Regex from scratch, the characters you might need and some larger blocks for the problem are already provided. Note that you do not need to use all of them; some are distractors.
If you are not sure of the meanings of the symbol, hover on a block to see the tooltip of its description.

To form your regex using the blocks, simply click on one block to add it to the end of the input, or drag one block to add it to anywhere of the input.

To remove a block in the input area, simply click on it and it will be removed.

Let's try solving a sample problem again. In this sample problem, please write a regular expression to match "abc", and not match "def".

Note that because this is a tutorial problem, it will not take you to the next problem right after you pass all the testcases. When you have done exploring the tool, click "Proceed to the questions" at the end of this page to move on to the next page of tutorial.
